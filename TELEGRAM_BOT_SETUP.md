## Создание бота в Telegram

### Шаг 1: Найдите @BotFather

1. Откройте приложение **Telegram**
2. В строке поиска введите: `@BotFather`
3. Выберите официальный аккаунт с галочкой верификации

### Шаг 2: Создайте нового бота

1. Нажмите кнопку **"Start"** или отправьте команду `/start`
2. Отправьте команду `/newbot` для создания нового бота
3. BotFather попросит вас ввести:
   - **Имя бота** (отображаемое имя): например, `TeacherTools Bot`
   - **Username бота** (уникальное, должно заканчиваться на "bot"): например, `teachertools_school_bot`

### Шаг 3: Получите токен бота

После успешного создания бота BotFather отправит вам сообщение с **токеном бота**:

```
Done! Congratulations on your new bot. You will find it at t.me/teachertools_school_bot.

Use this token to access the HTTP API:
123456789:ABCdefGHIjklMNOpqrsTUVwxyz1234567890
```

⚠️ **Важно:** Сохраните этот токен в безопасном месте! Он понадобится для настройки приложения.

### Шаг 4: Настройте описание бота (опционально)

Отправьте следующие команды BotFather для настройки бота:

```
/setdescription - Установить описание бота
/setabouttext - Установить текст "О боте"
/setuserpic - Установить аватарку бота
```

**Пример описания:**
```
Бот для управления учебным процессом. Просматривайте расписание, 
управляйте группами и студентами прямо из Telegram!
```

## Настройка приложения

### Шаг 1: Добавьте токен в .env файл

Откройте файл `.env` в корне проекта и добавьте строку:

```env
TELEGRAM_BOT_TOKEN=123456789:ABCdefGHIjklMNOpqrsTUVwxyz1234567890
```

Замените значение на ваш реальный токен, полученный от BotFather.

### Шаг 2: Установите зависимости

Убедитесь, что установлена библиотека `python-telegram-bot`:

```bash
pip install python-telegram-bot==20.7
```

Или установите все зависимости из requirements.txt:

```bash
pip install -r requirements.txt
```

### Шаг 3: Обязательные переменные окружения

Проверьте, что в `.env` файле присутствуют следующие переменные:

```env
# Telegram Bot
TELEGRAM_BOT_TOKEN=your_bot_token_here

# Database (пример для SQLite)
DATABASE_URL=sqlite:///instance/database.db

# Flask
SECRET_KEY=your_secret_key_here
```

## Настройка базы данных

### Модель TeacherTelegram

Модель `TeacherTelegram` уже создана в файле [`telegram_bot.py`](telegram_bot.py:48) и автоматически создаёт таблицу при запуске бота.

**Структура таблицы `teacher_telegram`:**

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | Integer | Первичный ключ |
| `teacher_id` | Integer | ID преподавателя (внешний ключ) |
| `telegram_chat_id` | BigInteger | ID чата Telegram |
| `telegram_username` | String | Username в Telegram |
| `first_name` | String | Имя пользователя |
| `last_name` | String | Фамилия пользователя |
| `is_active` | Boolean | Активна ли связь |
| `created_at` | DateTime | Дата создания |
| `updated_at` | DateTime | Дата обновления |

### Как работает привязка

1. Преподаватель получает свой `chat_id` через команду `/start` в боте
2. В веб-интерфейсе teacherTools преподаватель добавляет свой `chat_id` в настройках
3. Система создаёт запись в таблице `teacher_telegram`, связывая `teacher_id` с `telegram_chat_id`
4. Бот проверяет наличие связи при каждом запросе и предоставляет доступ только авторизованным преподавателям

## Привязка аккаунта преподавателя

### Шаг 1: Получите ваш Telegram Chat ID

1. Откройте созданного бота в Telegram (найдите по username, например `@teachertools_school_bot`)
2. Отправьте команду `/start`
3. Бот ответит сообщением:

```
👋 Добро пожаловать в teacherTools!

❌ Ваш Telegram аккаунт не привязан к системе.

🔑 Для привязки аккаунта:
1. Войдите в веб-приложение teacherTools
2. Перейдите в раздел настроек мессенджеров
3. Добавьте ваш Telegram ID

📋 Ваш Telegram ID: 123456789
```

**Скопируйте ваш Telegram ID** (цифры в последней строке).

### Шаг 2: Привяжите аккаунт в веб-приложении

1. Войдите в **teacherTools** через веб-браузер
2. Перейдите в раздел **"Настройки"** или **"Профиль"**
3. Найдите раздел **"Мессенджеры"** или **"Telegram"**
4. Введите ваш **Telegram ID** (полученный на шаге 1)
5. Сохраните настройки

### Шаг 3: Проверьте привязку

1. Вернитесь в Telegram и снова отправьте `/start`
2. Бот должен ответить:

```
👋 Здравствуйте, username!

✅ Ваш Telegram аккаунт уже привязан к системе.

📋 Доступные команды:
/help - Показать все команды
/today - Расписание на сегодня
...
```

Если вы видите это сообщение — привязка работает корректно! ✅

## Доступные команды бота

### Общие команды

| Команда | Описание |
|---------|----------|
| `/start` | Начало работы, проверка привязки аккаунта |
| `/help` | Показать справку по всем командам |

### Расписание

| Команда | Описание |
|---------|----------|
| `/today` | Показать расписание на сегодня |
| `/schedule ДАТА` | Показать расписание на конкретную дату (формат: `ГГГГ-ММ-ДД`) |

### Управление группами

| Команда | Описание |
|---------|----------|
| `/groups` | Список всех групп преподавателя |
| `/addgroup НАЗВАНИЕ КУРС ФОРМА` | Создать новую группу |

### Управление студентами

| Команда | Описание |
|---------|----------|
| `/students ГРУППА` | Список студентов в группе |
| `/addstudent ГРУППА ФИО [EMAIL]` | Добавить студента в группу |

### Управление занятиями

| Команда | Описание |
|---------|----------|
| `/addlesson ГРУППА ДАТА НАЧАЛО КОНЕЦ ТЕМА [АУДИТОРИЯ]` | Добавить занятие |
| `/deletelesson ID` | Удалить занятие по ID |

- `2024-02-15` — дата (ГГГГ-ММ-ДД)

## Устранение неполадок

### Бот не отвечает

**Проблема:** Бот не реагирует на команды

**Решения:**
1. Проверьте, что бот запущен:
   ```bash
   # Проверьте логи приложения
   tail -f app.log
   ```

2. Убедитесь, что токен правильный в `.env`:
   ```env
   TELEGRAM_BOT_TOKEN=your_correct_token_here
   ```

3. Перезапустите приложение:
   ```bash
   python app.py
   ```

### Ошибка "Вы не авторизованы"

**Проблема:** Бот отвечает "❌ Вы не авторизованы"

**Решения:**
1. Убедитесь, что вы правильно привязали Telegram ID в настройках
2. Проверьте, что используете правильный `chat_id` (не путайте с username)
3. Убедитесь, что запись в базе данных активна (`is_active = true`)

### Ошибка "Группа не найдена"

**Проблема:** Бот не находит группу

**Решения:**
1. Проверьте правильность названия группы (регистр важен!)
2. Убедитесь, что группа принадлежит вам (проверьте через `/groups`)
3. Используйте точное название из списка групп

### Проверка работы бота

**Проверка через логи:**
```bash
# Поиск сообщений о боте в логах
grep -i "telegram" app.log
```

**Ожидаемые сообщения при успешном запуске:**
```
TeacherTelegram table created/verified
Telegram bot started in separate thread
Telegram bot started successfully
```

### Перезапуск бота

Если бот перестал работать:

1. Остановите приложение (Ctrl+C)
2. Убедитесь, что все переменные окружения настроены
3. Запустите приложение заново:
   ```bash
   python app.py
   ```

## Рекомендации по безопасности

### 🔐 Хранение токена

- **Никогда** не публикуйте токен бота в открытых репозиториях
- **Никогда** не отправляйте токен в сообщениях
- Храните токен только в файле `.env`
- Добавьте `.env` в `.gitignore`:
  ```
  .env
  *.env
  ```

### 👤 Доступ к боту

- Бот доступен **только привязанным преподавателям**
- Каждый преподаватель видит **только свои группы и данные**
- Привязка осуществляется через подтверждение `chat_id`

### 🛡️ Защита данных

- Бот имеет доступ только к данным через SQLAlchemy модели
- Все запросы выполняются в контексте Flask приложения
- Используйте HTTPS для веб-приложения
- Регулярно обновляйте зависимости:
  ```bash
  pip install --upgrade python-telegram-bot
  ```

### ⚠️ В случае компрометации токена

Если токен был скомпрометирован:

1. Немедленно отзовите токен через @BotFather:
   ```
   /revoke
   ```

2. Получите новый токен:
   ```
   /token
   ```

3. Обновите токен в файле `.env`

4. Перезапустите приложение
